# kakao-auth-sender
카카오톡 메시지 자동 전송기
파이썬을 사용하여 검색대상자를 검색하소 message.txt 파일의 내용을 전송함
- message.txt 에 발송할 문자 내용 작성
- 카카오톡 실행
- sendKakao.exe 실행
- 검색할 키워드 입력 (ex. [거래처])
- 시작할 대상자 인덱스 번호 (ex. 0번째 부터 전송)
- 보낼 대상자 마지막 인덱스 번호 (ex. 10명에게 보냄)
- 대기
- 완료
